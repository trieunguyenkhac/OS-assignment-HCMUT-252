🚀 AsynapRous

Hybrid Client-Server & Peer-to-Peer Chat System

✨ Features

🔐

Authentication

Secure session-based user authentication with HTTP Basic Auth support

🌐

Reverse Proxy

HTTP reverse proxy for routing requests to backend services

⚡

Non-blocking I/O

AsyncIO-based non-blocking communication for high throughput

💬

Group Chat

Broadcast messages to channels with server-based delivery

👥

P2P Messaging

Direct peer-to-peer communication using async TCP connections

🔗

REST API

RESTful API for chat operations and peer discovery

📐 Architecture

Backend: Python AsyncIO server with TCP/HTTP protocol handling
Frontend: HTML/CSS/JavaScript with 1-second polling for real-time updates
P2P Layer: Direct asyncio.open_connection() for peer-to-peer messaging
Channel Layer: Group messaging via server with concurrent peer delivery
